## 機能
- マークダウン記法によるテキスト装飾
- 思いつきを逃がさない素早い入力遷移
- 内容が確認できるリスト表示
- 色付きタグによる柔軟な分類とフィルター
- 年、月、日による柔軟な日付フィルター
- 全文検索
- 閲覧を快適にする直感的な操作性
- フォントや行間のテキストスタイリング
- データのインポートとエクスポート
- iCloud 同期による複数デバイスでの利用
